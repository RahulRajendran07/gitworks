> git init [for initialising git repository]
> git status [for listing all untracked files]
> git add -A [for adding all untracked files to staging area]
> git commit -m "commit message" [to commit added file to local repository]
> git clone [exact copy from remote repositiry to local workspace ]
> git pull [changes from remote repository to local workspace]
> git push [local repository changes to remote repository]
> git config --list [to list configuration]
> git config --global user.name "username" [to add username]
> git config --global user.email "useremail" [to add user]
> git remote add remote_name remote_path [to add remote repository]
> git push remote_name branch_name [to push changes from local repository to remote repository]
